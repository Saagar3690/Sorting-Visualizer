# Sorting-Visualizer

Built using Python Tkinter

Implemented Algorithms:
- Bubble Sort
- Insertion Sort
- Selection Sort
- Merge Sort (iterative and recursive)
- Quicksort
- Shellsort
- Heapsort
- Comb Sort
- Cocktail Sort
- Monkey Sort
